## 命名規則
1.　半角アルファベット、``_``アンダースコア、``$``ダラーを入れるものにする。  
2.　変数名は数字から始められない。  
3.　予約ごと被るものは利用できない。 

```
const $; // OK $
const _title; // OK _
const jquery; // OK 小文字 
const TITLE; // OK 大文字
const es2015; // OK 先頭以外に数字がついたらいい  
const 日本語の命名: // 日本語はできる。  
```

```
const 1st; // 数字から始まるのはNG  
const 124; // 数字のみはNG
```
